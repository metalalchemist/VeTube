# VeTube
Čti si přístupným způsobem chaty u živých streamů (vlastních nebo svých oblíbených tvůrců) na YouTube, Twitchi, TikToku nebo QuentinC Playroomu.
## Funkce

- Automatický režim: Nech si číst nové zprávy v chatu v reálném čase svým oblíbeným SAPI 5 hlasem
- Virtuální rozhraní: Ovládej chaty jednoduchými klávesovými příkazy z kteréhokoli okna. K tomu potřebuješ spuštěný odečítač
- Podporované odečítače:
  - NVDA
  - JAWS
  - Window-Eyes
  - SuperNova
  - System Access
  - PC Talker
  - ZDSR
- Program si můžeš přizpůsobit a nastavit podle sebe:
  - Přepínání zvuků programu
  - Přepínání automatického režimu
  - Nastavení kategorií zpráv, které chceš vidět ve virtuálním rozhraní
  - Nastavení parametrů Sapi hlasu
- Snadné nastavení režimu čtení chatu: Můžeš si vybrat, jestli chceš číst všechny chaty, nebo jenom oznámení o příspěvcích a nových odběratelích
- Ukládání oblíbených streamů; Pročti si chat, kolikrát budeš chtít, aniž bys musel/a znovu zadávat odkaz
- Překládání zpráv v chatu z jednoho jazyka do druhého

## Klávesové zkratky

### Virtuální rozhraní
|Akce |Klávesová zkratka |
| ------------------------- | ------------- |
| Ztlumit hlas Sapi      | control+p           |
| Předchozí kategorie      | alt shift šipka doleva            |
| Následující kategorie      | alt shift šipka doprava            |
| Předchozí položka      | alt shift šipka nahoru           |
| Následující položka      | alt shift šipka dolů           |
| První položka      | alt shift home           |
| Poslední položka      | alt shift end           |
| Archivovat zprávu      | alt shift a           |
| Zkopírovat zprávu      | alt shift c           |
| Smazat kategorii      | alt shift d           |
| Přidat zprávu do oblíbených      | alt shift f           |
| Přepnout automatický režim      | alt shift r           |
| Přepnout zvuky programu      | alt shift p           |
| Prohledat zprávy v aktuální kategorii      | alt shift b           |
| Zobrazit vybranou zprávu v textovém poli      | alt shift v           |
| Spustit editor klávesových zkratek VeTube      | alt shift k           |
| Zapauzovat nebo odpauzovat přehrávání streamu      | control shift p           |
| Přetočit dopředu      | control shift šipka doprava           |
| Přetočit dozadu      | control shift šipka doleva           |
| Zvýšit hlasitost      | control shift šipka nahoru           |
| Snížit hlasitost      | control shift šipka dolů           |
